About

Welcome! This website uses NASA's Astronomical Image of the Day API in order to display beautiful images of space and all things related to space. THe "Random" tab allows you to query the database for a random astronomical image! You can find the latest images from NASA's API on the home page. For those interested, this site was built using the Flask web framework with Python along with MongoDB as the database, which stores the images with their details. You can find a link to my source code at the bottom of the page. THank you for stopping by and please enjoy!
